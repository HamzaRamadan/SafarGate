# Safar Gate - (بوابة سفر)

Welcome to **Safar Gate**, a progressive web application (PWA) designed to revolutionize the land transport market.

## Tech Stack
- **Framework**: Next.js (App Router)
- **UI**: React with TypeScript, Tailwind CSS, ShadCN/UI
- **Backend**: Firebase

## Key Features
- **Carrier**: Opportunity Hub, Dynamic Offers, Scheduled Trips.
- **Traveler**: Smart Search, Clear Decision Making, Secure Booking.
